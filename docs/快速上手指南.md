# Stellaris DLC Helper - 快速上手指南

> 三分钟学会使用本工具，解锁群星全部 DLC

---

## 💡 这个工具是干什么的？

一个帮你给群星游戏解锁所有 DLC 的小工具，简单来说就是：**点几下鼠标，所有 DLC 就能用了**。

---

## 🚀 怎么开始？

### 第一步：下载工具

1. 去 [Releases 页面](https://github.com/sign-river/Stellaris-DLC-Helper/releases/latest) 下载最新版本
2. 下载后解压到任意文件夹（建议放在容易找到的地方）
3. 双击 `Stellaris_DLC_Helper.exe` 运行

**注意：** 这个工具只支持 Windows 系统！

---

## 📖 使用教程

### 第一次使用

#### 1. 运行程序

- 双击 exe 文件启动程序
- 第一次运行会弹出更新检查和公告（如果有的话）

#### 2. 选择游戏目录

- 程序会自动尝试找到你的群星游戏安装位置
- 如果没找到，点击"浏览"按钮，手动选择游戏文件夹
  - 通常在：`C:\Program Files (x86)\Steam\steamapps\common\Stellaris`

#### 3. 点击"一键解锁"

- 点一下，剩下的交给程序自动完成
- 它会自动：
  - 测试最快的下载源
  - 下载所有 DLC 文件
  - 安装 DLC
  - 打上解锁补丁

#### 4. 等待完成

- 下载过程中会显示实时速度和进度
- 下载完成后会自动安装
- 看到"操作完成"提示就好了

#### 5. 启动游戏

- 正常通过 Steam 启动群星游戏
- 进入游戏后就能看到所有 DLC 了

---

## 🎮 日常使用

### 游戏更新了怎么办？

1. 打开工具
2. 再次点击"一键解锁"
3. 程序会自动下载新版本的 DLC
4. 完成后就能玩了

### 下载太慢怎么办？

程序会自动选择最快的下载源，一般情况下：

- **有梯子**：会使用 R2 或 GitHub，速度很快
- **没梯子**：会使用国内服务器，速度也还行
- **都不快**：会使用 Gitee，保底 2-3 MB/s

#### 手动测速

- 点击"设置"按钮
- 切换到"测速"选项卡
- 点击"开始测速"查看各个源的速度

### 暂停/继续下载

- 下载过程中可以点击"暂停下载"
- 想继续下载时点击"继续下载"
- 支持断点续传，不用担心网络中断

## 🔧 工具按钮说明

| 按钮         | 作用                                        |
| ------------ | ------------------------------------------- |
| **一键解锁** | 自动完成所有操作：下载→安装→打补丁          |
| **浏览**     | 手动选择游戏安装目录                        |
| **设置**     | 打开设置窗口，可以测速、查看日志、清理缓存  |
| **还原游戏** | 把游戏恢复到原始状态（移除所有 DLC 和补丁） |

| **继续下载** | 继续之前暂停的下载                          |

## ❓ 常见问题

### Q1: 第一次用，需要准备什么？

A: 什么都不用准备！只需要：

- Windows 电脑
- 已经安装了《群星》游戏（Steam 版）
- 能上网

### Q2: 会不会封号？

A: 这是一个本地工具，只修改你电脑上的游戏文件，**不涉及 Steam 账号**，所以不会封号。

### Q3: 下载的文件保存在哪里？

A: 默认保存在工具目录下的 `Stellaris_DLC_Cache` 文件夹里。

### Q4: 怎么知道 DLC 装成功了？

A: 启动游戏后：

- 进入游戏主菜单
- 点击"内容"或查看 DLC 列表
- 应该能看到所有 DLC 都显示可用

### Q5: 工具下载的是什么？

A: 下载的是：

- DLC 的资源文件（.dlc 文件）
- CreamAPI 补丁（用于解锁 DLC）
- 游戏必需的配置文件

### Q6: 想要卸载怎么办？

A:

1. 在工具里点击"还原游戏"，把游戏恢复原样
2. 删除工具文件夹就行

### Q7: 提示"找不到游戏目录"？

A:

- 点击"浏览"按钮
- 找到 Stellaris 游戏的安装文件夹
- 正确的文件夹里应该有 `stellaris.exe` 文件

### Q8: 下载失败或者出错了？

A:

1. 检查网络连接
2. 尝试在"设置→测速"里手动测速，换个源
3. 点击"一键解锁"重新试一次
4. 如果还是不行，到 [Issues](https://github.com/sign-river/Stellaris-DLC-Helper/issues) 反馈

### Q9: 工具会自动更新吗？

A: 会的！

- 每次启动都会检查更新
- 有新版本会提示你
- 可以选择更新或跳过

### Q10: 占用多少空间？

A:

- 工具本身：约 20 MB
- DLC 缓存：约 80-100 MB（可以在设置里清理）
- 总共：约 120 MB

## 📝 工作原理简述

这个工具做了三件事：

1. 下载 DLC 文件：从服务器下载官方的 DLC 资源文件
2. 安装 DLC：把文件放到游戏目录的正确位置
3. 应用补丁：使用 CreamAPI 让游戏认为你拥有这些 DLC

全程自动，不需要你手动操作。
