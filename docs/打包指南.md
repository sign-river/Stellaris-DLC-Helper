### 概述
本项目通过 PyInstaller 打包为 Windows exe，推荐使用自动化 `build.py` 创建虚拟环境、安装依赖并构建。

### 快速打包
自动打包：
```
python build.py
```

手动打包步骤:
1. 创建虚拟环境 `python -m venv Stellaris_DLC_Cache/venv/build_venv`
2. 安装依赖 `pip install -r requirements-build.txt`
3. 运行 PyInstaller `pyinstaller Stellaris-DLC-Helper.spec`

### 注意点
- 需要 Python 3.7+
- 推荐在干净环境中打包
- 若缺少 DLL 或模块，可通过 `hiddenimports` 补充
- `pairings.json`（用于 GitHub/Gitee 映射）会被包含在最终发布包中（与 `config.json` 一起复制），如果需要在不重建 exe 的情况下更新映射文件，可直接替换此文件。

### 清理建议
打包时产生的临时文件可通过脚本清理，以节省磁盘空间。
