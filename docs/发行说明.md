# 发行说明（v1.x.x）

本文件基于原英文发布说明（英文原文已删除），列出版本 v1.0.0 的发布说明、特性、已知问题和兼容性更新。

## v1.0.0（初始发布）
- 自动识别游戏目录并加载DLC列表
- 支持多源下载（R2、GitHub、国内云、Gitee）
- 本地缓存、智能断点续传、进度显示与还原功能
- 中文GUI，支持中文路径（增强兼容性）
 
## 已知问题
- 某些杀毒软件可能误报
- 网络不稳定时可能需要多次重试
