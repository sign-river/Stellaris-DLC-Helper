**配置管理说明**
- 位置：设置 → 配置管理 选项卡。
- 作用：显示当前程序生效的 `config.json` 路径，便于诊断、复制或在资源管理器中打开该目录进行替换。
- 使用说明：
  - 如果你想替换配置文件，请先退出程序，然后将新的 `config.json` 放到 EXE 同目录（或你运行程序时的目录），再重新启动。程序优先读取 EXE 同目录的 `config.json`。
  - 如出现“仍然使用旧来源”的问题，请打开“设置 → 配置管理”，复制并发送 `config.json` 路径或将该目录下的 `runtime_info.txt` 与日志一起提交给开发者进行诊断。
- 备注：程序启动时也会在 EXE 目录和当前工作目录写入 `runtime_info.txt`，包含 `sys.executable`、`cwd`、`config_path`、`DLC_SERVER_URL` 等信息，便于快速定位问题。
