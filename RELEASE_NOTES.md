## ✨ 主要特性

### 🎮 游戏集成
- **专注群星** - 专门为Stellaris游戏优化的DLC管理工具
- **自动检测** - 无需手动输入AppID，程序自动识别游戏
- **智能识别** - 自动检测已安装的DLC，避免重复下载

### ☁️ 云端下载
- **高速下载** - 支持多线程下载和断点续传
- **智能重试** - 网络异常时自动重试，提升下载成功率
- **进度显示** - 实时显示下载进度和速度

### 💾 智能缓存
- **本地缓存** - 下载的DLC文件本地缓存，避免重复下载
- **节省流量** - 相同DLC只需下载一次
- **离线使用** - 缓存文件可离线安装

### 🔄 操作管理
- **操作记录** - 详细记录所有安装操作
- **一键还原** - 可随时恢复游戏原始状态
- **精确还原** - 基于操作日志的精确还原机制

### 🎨 用户体验
- **纯中文界面** - 简洁易用的中文图形界面
- **现代化设计** - 基于CustomTkinter的现代化UI
- **响应式布局** - 自适应窗口大小和分辨率

## 🔧 技术特性

### 系统要求
- **操作系统**: Windows 7/8/10/11 (64位)
- **内存**: 至少512MB可用内存
- **存储**: 约50MB可用磁盘空间
- **网络**: 需要互联网连接下载DLC

### 技术栈
- **Python**: 3.7+
- **UI框架**: CustomTkinter 5.2.0+
- **HTTP客户端**: requests 2.28.0+
- **图像处理**: Pillow 9.0.0+
- **打包工具**: PyInstaller 5.0+

## 📦 安装使用

### 下载安装
1. 前往 [Releases](https://github.com/sign-river/Stellaris-DLC-Helper/releases/latest) 页面
2. 下载 `Stellaris-DLC-Helper-v1.0.0.zip`
3. 解压到任意位置（建议非中文路径）
4. 运行 `Stellaris-DLC-Helper.exe`

### 基本使用
1. **选择游戏目录** - 点击"浏览"选择Stellaris安装目录
2. **查看可用DLC** - 程序自动加载DLC列表
3. **下载安装** - 勾选所需DLC，点击下载安装
4. **还原游戏** - 如需移除DLC，点击"还原游戏"
